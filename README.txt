This project calculates the statistical parameters for samples stored in the input file. Please note that it currently is capable to analyze two independent samples only.

1. Unzip the files in the target folder
2. Ensure python version 3.0 and above is installed in the computer
3. Ensure Numpy is installed in the computer
3. Ensure the computer is connected to the internet
4. Run the StatisticalInference.py either through the command line or a developer environment
